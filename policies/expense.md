# 核銷制度

## 基本原則

工作相關支出，憑發票或收據向公司申請核銷。

## 流程

1. 保留發票或收據（需有品項說明）
2. 填寫費用申請單，說明用途
3. 送交 Ronny 確認
4. 每月統一結算，隨薪資發放

## 可核銷項目

| 類別 | 說明 |
|------|------|
| 交通費 | 因公出差、會議交通 |
| 餐費 | 因公會議、工作餐（需說明對象與用途） |
| 設備耗材 | 工作所需文具、耗材 |
| 軟體工具 | 工作所需訂閱服務（需事先告知） |
| 其他 | 經 Ronny 事先同意之支出 |

## 注意事項

- 個人消費不予核銷
- 超過 NT$5,000 的單筆支出，請事先確認再購買
- 無統一編號發票仍可核銷，但盡量取得載有品項的收據
