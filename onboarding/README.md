# 新人入職指南

歡迎加入歐噴！以下是入職第一週需要完成的事項。

## 第一天

- [ ] 取得 Google Workspace 帳號（@openfun.tw），由 Ronny 建立
- [ ] 加入公司 Google Calendar，確認可看到公共行事曆
- [ ] 加入公司 GitHub 組織：[openfunltd](https://github.com/openfunltd)
- [ ] 確認 Slack / 主要溝通工具已可使用（待補：確認目前使用的溝通工具）

## 第一週

- [ ] 閱讀本員工手冊
- [ ] 了解[請假制度](../policies/leave.md)與[核銷制度](../policies/expense.md)
- [ ] 確認工作所需設備（如需採購請告知 Ronny）
- [ ] 了解負責的專案與工作內容，參見[專案文件](../projects/README.md)
- [ ] 了解技術環境，參見[技術文件](../tech/README.md)
